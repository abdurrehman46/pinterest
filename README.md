# 📌 Pinterest Clone (HTML & CSS)

A responsive **Pinterest-inspired web layout** built using **HTML and CSS**.  
This project recreates the Pinterest-style image grid layout to practice modern layout techniques like **CSS Grid** and **Flexbox**.

---


## 🛠️ Technologies Used  

- HTML5  
- CSS3  
- CSS Grid  
- Flexbox  

---

## ✨ Features  

- Pinterest-style grid layout  
- Responsive design  
- Modern UI layout  
- Hover effects  
- Clean and simple design  
- Mobile-friendly layout
